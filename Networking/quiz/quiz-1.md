# Quiz 1 – Networking Basics (Review)

---

### Q1. Which OSI layer handles IP addressing and routing?

**Answer:** Network  
**Explanation:** The Network layer is responsible for logical addressing (IP) and routing packets between networks.

---

### Q2. What is the main purpose of a MAC address?

**Answer:** Identify a device within a local network  
**Explanation:** A MAC address uniquely identifies a device at the Data Link layer (LAN level).

---

### Q3. What is a key advantage of IPv6 over IPv4?

**Answer:** Larger address space  
**Explanation:** IPv6 provides vastly more IP addresses, solving IPv4 exhaustion.

---

### Q4. Which protocol guarantees reliable data delivery?

**Answer:** TCP  
**Explanation:** TCP ensures ordered, error-checked, and reliable delivery.

---

### Q5. Which protocol is faster but does NOT guarantee delivery?

**Answer:** UDP  
**Explanation:** UDP is faster because it skips error checking and retransmission.

---

### Q6. What does a CNAME record do?

**Answer:** Maps a domain to another domain name  
**Explanation:** CNAME points one domain name (alias) to another.

---

### Q7. What is the difference between HTTP and HTTPS?

**Answer:** HTTPS uses encryption, HTTP does not  
**Explanation:** HTTPS adds TLS/SSL encryption for secure communication.

---

### Q8. What is SSH mainly used for?

**Answer:** Secure remote login to a system  
**Explanation:** SSH provides encrypted command-line access to remote machines.

---

### Q9. What is NAT primarily used for?

**Answer:** Translating private IP addresses to public IP addresses  
**Explanation:** Network Address Translation (NAT) allows multiple devices to share one public IP.

---

### Q10. Which protocol is correctly matched to its default port?

**Answer:** SSH → 22  
**Explanation:** Common ports: HTTP (80), HTTPS (443), SSH (22), DNS (53).
